Installation
------------

You can add the **RedKiteCmsBaseBlocksBundle** to your application adding it to the composer.json
file of your application:

.. code-block:: text

    {
        "require": {
            "redkite-apps/redKite-cms-base-blocks-bundle": "dev-master"
        }
    }
